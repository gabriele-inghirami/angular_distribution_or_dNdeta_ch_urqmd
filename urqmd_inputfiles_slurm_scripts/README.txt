inputfile200, inputfile900, inputfile2760, inputfile7000: UrQMD config files for pp collisions (angular distribution study)
inputfile200_pp, inputfile900_pp, inputfile2760_pp, inputfile7000_pp: UrQMD config file for pp collisions (dN/deta_ch study)
inputfile_pPb, inputfile_PbPb: UrQMD config files for p-Pb and PbPb collisions (dN/deta(eta) study)
inputfile_Au_200: UrQMD config file for central Au+Au at Ecm 200 GeV (dN/deta(eta) study)
inputfile_5020_PbPb: UrQMD config file for perfectly central Pb+Pb at Ecm 5.02 TeV, t=0.5fm, no decays, (dN/dy interacting/non inter. study)
launch_runs_urqmd_5TeV.sh launch_runs_urqmd_dNdeta_200.sh: bash scripts that call the slurm scripts (dN/deta(eta) study)
run_urqmd_5TeV.bash and run_urqmd_dNdeta_200.bash: slurm scripts (dN/deta(eta) study, for PbPb 5.02TeV and AuAu 200GeV)
launch_run_urqmd_5TeV_beginning.sh: bash scripts that calls the slurm script (dN/y interacting vs non interacting study)
run_urqmd_5TeV_beginning.bashh: slurm script (dN/dy interacting - non interacting study, for PbPb 5.02TeV and AuAu 200GeV)
launch_runs_urqmd_pp.sh: bash script that calls the slurm script run_urqmd_pp.sh (dN/deta for charged particles in pp collisions at 200,900,2760 and 7000 GeV)
run_urqmd_pp.sh slurm script (dN/deta for charged particles in pp collisions at 200,900,2760 and 7000 GeV)
